  • Project date: 01 January, 2026

Project Overview

This project demonstrates how automation and data analytics can be combined to analyze job market trends by integrating web scraping, data processing, and visualization tools, the system provides meaningful insights into hiring patterns and required skills in the job market.

  • What the project does
    • Automatically scrapes job listing details using Selenium.
    • Stores the extracted data in a structured table format.
    • Generates a CSV dataset automatically and sends it via email after scraping is completed.
    • Processes, analyzes, and predicts the dataset using Python.
    • Created an interactive dashboard to visualize job market insights.

  • Dashboard Insights
    • Total job listings
    • Job distribution by type (Full-time / Part-time)
    • Jobs by location
    • Top hiring companies
    • Predicted Skils
    • Skils Missing
    • Most in-demand skills
    • Job posting trends over time
